- Virtual DOM is nothing but a logical DOM or JavaScript DOM.
- It will works at application level itself, so it will occupy entire memory space in the application.
- It will check the previous state with new state in the component tree structure.
- If any changes are there, then only It will re-load only updated component state.
- So it will consume less memory but it will improves high perform.
Virtual DOM uses Diffie algorithm, Diffie algorithm which component changed data details and send the details to Virtual DOM.
Virtual DOM Immediately send the changes to Real DOM (Browser DOM).
Real DOM (Browser DOM) where changes is happened in state that components only re-rendering remaining componenets did not re-rendering that is virtual DOM benefit.

- Real DOM is also called as physical DOM.
- It will works at the browser level, every time it will create new memory instance even for small changes in the web application.
- It will reload entire webapplication at a time so consumes more memory space and gives poor performance.

- JSX is an Javascript XML or an extension to the JavaScript.
- JSX is nothing but a combination of.
1. JavaScript.
2. CSS.
3. XML (Custom Tags).
4. HTML.

- Component is a piece of code or fundemental building block of a React application.
- Every React Application contains at-least one root component and n-number of child componenets
- All componenets will form like as tree structure in the browser at the time of running a react application.
- Every React component can have their own structure, methods as well as API's, means each component will work independently from one another but all child components merge in a parent component, which will be the final UI of your application.
Advantange of Components
- Components will make easy to build UI
- If anything goes wrong in UI, It is very easy to findout/debug any issue.
- They can reusable as per your need.
